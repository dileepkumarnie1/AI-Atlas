## âš¡ Performance Tips

### GPU Acceleration
All animations use `transform` properties:
- `translateY()` for floating
- `scale()` for hover effects
- `rotate()` for sparkles

### Efficient Timing
- Background: 15s (slow, smooth)
- Particles: 6s (medium speed)
- Icons: 2s (noticeable but not jarring)
- Sparkles: 2s (eye-catching)
